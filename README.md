## Setup

```bash
npm install
gulp build:watch
```

## Usage

The app recognizes bulleted or numbered lists in a format like the following:

```
1. Red
  a. 9 to 12:15
    i. Did this
2. Tyme
  a. 12:45 to 5
    i. Did that
```
